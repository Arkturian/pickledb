Welcome

pickleDB is a lightweight and simple key-value store. It is built upon Python's json module and was inspired by redis. It is licensed with the BSD three-caluse license.

pickleDB is Fun

>>> import pickledb as db

>>> db.load('example.db')

>>> db.set('key', 'value')
True

>>> db.get('key')
'value'

And Easy to Install

$ pip install pickledb

pickleDB got its name from Python's "pickle" module, which it used to use. However, now pickleDB uses the "json" module. It is faster and cleaner. But the name stuck!
