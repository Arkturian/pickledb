About

pickleDB is a lightweight, simple, and fast key-value store. It is built upon Python's json module and was inspired by redis. It is licensed with the BSD three-caluse license.

Installation

The easiest way to download and install pickleDB on your system is to use pip: 
$ sudo pip install pickledb

Examples

The following is a run through of the most basic of pickleDB's commands. 
>>> import pickledb as db

>>> db.load('test.db')
True

>>> db.set('key', 'value')
True

>>> db.get('key')
'value'

>>> db.rem('key')
True

>>> db.get('key')
KeyError: 'key'

>>> db.lcreate('a list')
True

>>> db.ladd('a list', 'something')
True

>>> db.ladd('a list', 'now something else')
True

>>> db.getall('a list')
['something', 'now something else']

>>> db.lget('a list', 0)
'something'

>>> db.lget('a list', 1)
'now something else'

>>> db.lpop('a list', 0)
True

>>> db.lgetall('a list')
['now something else']

>>> db.lrem('a list')
True

>>> db.lgetall('a list')
KeyError: 'a list'
